## Overview

A clean analytics dashboard that aggregates social media metrics into a single view. Built to practice data visualization and API integration patterns.

## Features

- Multi-platform metric aggregation
- Interactive charts and trend visualizations
- Date range filtering
- Dark theme UI matching platform aesthetics

## Tech Stack

Next.js for the framework, chart libraries for data visualization, and REST API integrations for pulling platform data.
