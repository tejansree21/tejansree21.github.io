## Overview

Cost Rental Finder helps users search for and compare affordable rental properties available through Ireland's cost rental scheme. The app pulls data from multiple sources and presents it in a clean, filterable interface.

## Features

- Search and filter properties by location, price, and type
- Interactive map view with property markers
- Real-time data updates via Supabase
- Mobile-responsive design

## Tech Stack

Built with React on the frontend, Supabase for the backend and real-time data layer, and integrated with mapping APIs for location visualization.
